# 🌱 Campo e Futuro

Projeto desenvolvido por Sofia utilizando HTML e CSS com o tema **"Agro Forte: Futuro Sustentável – Equilíbrio entre Produção e Meio Ambiente"**.

## 📌 Objetivo

Mostrar a importância da agricultura sustentável e como a produção de alimentos pode acontecer de forma responsável, preservando o meio ambiente.

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3



## 📂 Estrutura

```text
├── index.html
├── style.css
└── README.md
```

## 🌾 Tema

O projeto destaca a relação entre o agronegócio, a preservação ambiental e o desenvolvimento sustentável.
